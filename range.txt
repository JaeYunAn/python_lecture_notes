Range

* 연속된 정수 시퀀스를 생성하는, 변경 불가능한 자료형
* 주로 반복문과 함께 사용되어 특정 횟수만큼 코드를 반복 실행할 때 매우 유용
* 실제로 모든 숫자를 메모리에 저장하는 대신 시작 값, 끝 값, 간격이라는 규칙만 기억하여 메모리를 메우 효율적으로 사용

range(start, stop, step)
range()는 1개, 2개, 또는 3개의 매개변수(인자)를 가진다.

* 매개변수: 함수를 정의할 때, 함수가 받을 값을 나타내는 변수
* 인자: 함수를 호출할 때 실제로 전달되는 값

range(stop)
* 매개변수가 하나면 stop으로 인식
* start는 0이, step은 1이 기본값으로 자동 설정
* range(5) -> 0, 1, 2, 3, 4

range(start, stop)
* 매개변수가 두 개면 start와 stop으로 인식
* step은 1이 기본값으로 자동 설정
* range(2, 5) -> 2, 3, 4

range(star, stop, step)
* 모든 매개변수를 직접 지정
* range(2, 10, 2) -> 2, 4, 6, 8
