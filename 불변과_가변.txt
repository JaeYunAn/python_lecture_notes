불변 vs 가변

컬렉션 타입은 생성 후 내용을 변경할 수 있는지 없는지에 따라 '불변'과 '가변' 두 그룹으로 나뉨

불변: 변경 불가, 안정성, 예측 가능 -> str, tuple, range
가변: 변경 가능, 유연성, 효율성 -> list, dict, set

불변
my_str = 'hello'
my_str[0] = 'z'
# TypeError: 'str' object does not support item assignment

가변
my_list = [1, 2, 3]
my_list[0] = 100
print(my_list)
# [100, 2, 3]
